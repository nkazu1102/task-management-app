📁 OneDriveでタスク管理アプリを共有

✅ OneDriveが最適な理由
• 社内で既に使用している
• 権限管理が簡単
• 自動同期でファイル管理が楽
• 無料で利用可能

🎯 方法1: 共有フォルダ経由（推奨）

ステップ 1: フォルダの準備

  1. OneDriveで新しいフォルダを作成（例：「タスク管理アプリ」）
  2. publicフォルダの中身をすべてコピー
  3. OneDriveフォルダに貼り付け

ステップ 2: 共有設定

  1. フォルダを右クリック → 共有
  2. 特定のユーザー を選択
  3. 使用したいメンバーのメールアドレスを追加
  4. 権限を 表示 に設定（編集不要）
  5. 共有 ボタンをクリック

ステップ 3: 使用方法

  1. 各メンバーが受信した共有通知からフォルダにアクセス
  2. index.html をダブルクリック
  3. ブラウザで開く（Edgeなど）
  4. ブックマークに保存して今後使用

🌐 方法2: OneDrive Web経由

ステップ 1: Webアップロード

  1. onedrive.live.com にアクセス
  2. フォルダを作成してファイルをアップロード
  3. フォルダを共有設定

ステップ 2: アクセス方法

  1. 共有リンクから OneDrive Web にアクセス
  2. index.html をクリック
  3. アプリで開く → ブラウザ を選択
⚠️ 注意点
• データは各ユーザーのブラウザに個別保存されます
• OneDriveのファイル自体は読み取り専用にしてください
• 初回アクセス時はセキュリティ警告が出る場合があります

🔧 トラブルシューティング

JavaScriptが実行されない場合:

  • ブラウザの設定で JavaScript を有効にする
  • Internet Explorer の代わりに Edge や Chrome を使用
  • ファイルをローカルにダウンロードしてから開く

ファイルが開けない場合:

  • ファイルを右クリック → プログラムから開く → ブラウザ
  • ファイルの拡張子が .html になっているか確認

🚀 推奨設定

  1. 管理者: OneDriveの共有フォルダを作成・管理
  2. メンバー: 表示権限でアクセス、各自ブックマーク保存
  3. 更新: 管理者がファイルを更新すると全員に反映
