📋 タスク管理アプリ - 全員使用ガイド

⚠️ 重要
このアプリは各ユーザーのブラウザにデータを保存します。
共有データが必要な場合は、サーバー版の検討をお勧めします。

🌐 方法1: Netlify（最も簡単・推奨）

✅ メリット: 無料、簡単、即座に使用可能
❌ デメリット: データは個人別

手順:

  1. netlify.com でアカウント作成
  2. 「Add new site」→「Deploy manually」
  3. publicフォルダをドラッグ&ドロップ
  4. 生成されたURLを全員に共有

📁 方法2: 社内共有フォルダ

✅ メリット: 社内のみ、IT部門不要
❌ デメリット: ネットワークドライブアクセス必要

手順:

  1. publicフォルダを社内共有フォルダにコピー
  2. 各ユーザーが \\server\shared\taskapp\index.html をブラウザで開く
  3. ブックマークに登録推奨

🖥️ 方法3: 社内Webサーバー

✅ メリット: 社内専用、高速
❌ デメリット: IT部門の協力必要

手順:

  1. IT部門に依頼してWebサーバーを設定
  2. publicフォルダの内容をWebサーバーにアップロード
  3. 社内URL（例：http://intranet/taskapp/）でアクセス

☁️ 方法4: その他クラウドサービス

  • Vercel: Netlify同様に簡単
  • GitHub Pages: GitHubアカウント必要
  • Firebase Hosting: Googleアカウントで利用可能

💾 データ共有について

現在の仕様では、各ユーザーのデータは個別のブラウザに保存されます。

共有データが必要な場合:

  • データベース連携版の開発が必要
  • Excel等でのデータエクスポート・インポート機能の追加
  • 定期的なデータ同期ツールの導入

🚀 推奨手順

  1. Netlifyで即座に全員使用開始
  2. 使用感を確認・フィードバック収集
  3. 必要に応じて共有データ機能を検討
